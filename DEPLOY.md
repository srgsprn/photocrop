# Деплой бота на сервер (работа 24/7)

Пока бот запущен на твоём ноутбуке — он работает. Закрыл ноутбук или выключил — бот перестаёт отвечать. Чтобы он работал **автономно**, его нужно развернуть на облачном сервисе.

Ниже — два простых варианта: **Railway** и **Render**. Оба дают бесплатный или недорогой тариф и подходят для этого бота.

---

## Вариант 1: Railway (проще всего)

1. Зайди на [railway.app](https://railway.app), войди через GitHub.
2. **New Project** → **Deploy from GitHub repo** → выбери репозиторий **photocrop**.
3. Root Directory оставь пустым (весь репозиторий — это бот).
4. **Variables** → добавь переменную:
   - `BOT_TOKEN` = твой токен от BotFather.
5. **Settings** → **Deploy**:
   - **Start Command**: `python bot.py`  
   Или оставь пустым, если используешь Procfile с `worker: python bot.py`.
6. Деплой запустится. Первый запуск может занять 2–3 минуты (скачивание модели rembg).

Готово: бот будет работать постоянно, пока проект на Railway запущен.

---

## Вариант 2: Render

1. Зайди на [render.com](https://render.com), войди через GitHub.
2. **New** → **Background Worker**.
3. Подключи репозиторий **photocrop**. Root Directory оставь пустым.
4. **Build Command**: `pip install -r requirements.txt`  
   **Start Command**: `python bot.py`
5. **Environment** → добавь переменную `BOT_TOKEN` со значением токена.
6. **Create Background Worker**.

На бесплатном тарифе воркер может «засыпать» после периода неактивности; при первом сообщении боту может понадобиться 30–60 секунд на пробуждение. Платный тариф держит бота всегда включённым.

## Про Vercel

**Vercel** рассчитан на короткие HTTP-запросы (серверлесс), а не на постоянно работающий процесс. Чтобы бот работал на Vercel, нужно переводить его на **webhooks** (Telegram шлёт запросы на твой URL). Плюс rembg тяжёлый (модель ~176 МБ, много памяти и времени на «холодный» старт), из‑за чего на бесплатном тарифе Vercel возможны таймауты и лимиты памяти. Для этого бота удобнее **Railway** или **Render**.

## Бесплатные альтернативы (без платного тарифа)

Если Railway или Render просят оплату или закончился бесплатный период, можно использовать сервисы с реально бесплатным тарифом:

### Fly.io (бесплатный лимит) — пошагово

**Где ставятся зависимости:** при сборке образа на Fly (в `Dockerfile` уже есть `pip install -r requirements.txt`). Ничего ставить вручную не нужно.

**Где задать `BOT_TOKEN`:** в Fly.io переменные окружения для бота задаются как **Secrets**. Два способа:

1. **В терминале** (после `fly launch`):
   ```bash
   cd /путь/к/photocrop
   fly secrets set BOT_TOKEN=сюда_вставь_токен_от_BotFather
   ```
2. **В веб-панели:** [fly.io/dashboard](https://fly.io/dashboard) → выбери приложение **photocrop** → **Secrets** → **Set secret** → имя `BOT_TOKEN`, значение — твой токен.

**Шаги с нуля:**

1. Установи [flyctl](https://fly.io/docs/hands-on/install-flyctl/) и войди: `fly auth login`.
2. В папке с ботом (где лежат `bot.py`, `Dockerfile`, `fly.toml`):
   ```bash
   cd /Users/monstrik/Desktop/photocrop
   fly launch --no-deploy
   ```
   Если спросит имя приложения — можно оставить `photocrop` или ввести своё. Регион — любой ближайший (например `ams`).
3. Задай токен:
   ```bash
   fly secrets set BOT_TOKEN=твой_токен_от_BotFather
   ```
4. Деплой:
   ```bash
   fly deploy
   ```
   Сборка образа поставит все зависимости из `requirements.txt`; первый запуск может занять 2–3 минуты (скачивание модели rembg).

В репозитории уже есть `Dockerfile` и `fly.toml` — Fly сам соберёт образ и запустит бота.

### Oracle Cloud Free Tier (всегда бесплатный VPS)

- **Сайт:** [oracle.com/cloud/free](https://www.oracle.com/cloud/free/)
- Бесплатно навсегда: 1–2 небольшие VPS (ARM Ampere), 24/7. Нужна карта для верификации (списаний нет в рамках free tier).
- Как: зарегистрироваться → создать VM (Ubuntu) → по SSH установить Python, клонировать репо **photocrop**, поставить зависимости, задать `BOT_TOKEN` и запустить `python bot.py` через `systemd` или `screen`/`tmux`, чтобы работало в фоне.

### Koyeb

- **Сайт:** [koyeb.com](https://www.koyeb.com)
- Есть бесплатный тариф для маленьких сервисов.
- Подключи GitHub-репо **photocrop**, укажи команду запуска `python bot.py`, добавь переменную `BOT_TOKEN`.

### Zeabur

- **Сайт:** [zeabur.com](https://zeabur.com)
- Бесплатный тариф для небольших проектов, похож на Railway.
- Подключи репо **photocrop**, добавь `BOT_TOKEN`, деплой.

### Очень дёшево (если бесплатные лимиты не подходят)

- **Hetzner** — VPS от ~4 €/мес, полный контроль, надёжно для бота 24/7.
